# n8n Dart Client

A Dart client package for interacting with the [n8n Public API](https://docs.n8n.io/api/).

Built and maintained by **Spinex IO**.

---

## 📦 Installation

Add this to your `pubspec.yaml`:

```yaml
dependencies:
  n8n_client:
    git:
      url: https://github.com/spinex-io/n8n_client.git
